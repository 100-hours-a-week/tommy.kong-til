## 차원
- 차원 : 사물이나 공간의 크기, 범위, 트징 등을 나타내는 **측정 가능한 속성**
  - 측정 가능 : 수로 표현할 수 있다. 
  - 속성 : 대상에 속한 성질 
- 차원의 정의 : 배열에서 데이터를 구성하는 축의 개수 
    - 스칼라 = 0차원 배열 
    - 벡터 = 1차원 배열 -> 축이 1개
    - 행렬 = 2차원 배열 -> 축이 2개
    - 텐서 = 3차원 배열 -> 축이 3개
  - 대상은 배열이고 속성은 차원이다.
